# VMON
A visual machine language monitor for the VIC20 and other 6502 based machines.

## Resources
* Project page: https://github.com/egnagflow/vmon
* Ready to run images: https://github.com/egnagflow/vmon/releases
  
## Examples
### PRG build for unexpanded VIC20
This capture shows VMON running on an unexpanded VIC20 (VICE) with 3KB of RAM. After start-up VMON runs the built-in example code in continuous single step mode.
